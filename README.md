# 📸 Automatic Selfie Capture

##Overview

This project is a Python-based application that automatically captures selfies using your webcam.
It uses computer vision techniques to detect a face and capture an image without manual input.

---

##Features

* 📷 Automatic face detection
* 🤖 Hands-free selfie capture
* ⚡ Fast and lightweight
* 🖥️ Works with webcam

---

##Technologies Used

* Python
* OpenCV
